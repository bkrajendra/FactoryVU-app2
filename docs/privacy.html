Overview

FactoryVU (“the app”) helps you monitor and manage industrial IoT sensors and devices. This policy explains what data the app processes and how it is used.

Data we collect

  • API key (stored on-device): The app stores your API key locally on your device to authenticate requests to your cloud endpoint.
  • Device data you view: Sensor readings (e.g., temperature/humidity) and device identifiers returned by your configured API endpoint.
  • Local network discovery metadata: If you use local discovery, the app may discover devices/services on your local network using Bonjour/mDNS (e.g., service name/type, host, port, and TXT records).

Data we do not sell

We do not sell your personal information.

How we use data

  • To authenticate to your configured cloud API using your API key
  • To load and display your devices and sensor readings
  • To discover devices available on your local network (if enabled by you)

Where data is stored

  • On your device: Your API key and app settings are stored locally (e.g., in local storage).
  • Cloud API: Device data is retrieved from the API endpoint configured by the app.

Permissions

The app may request access to your Local Network to discover devices via Bonjour/mDNS. You can allow or deny this permission at any time in iOS Settings.

Third-party services

If the app integrates third-party services (for example, push notification providers), those services may process data according to their own privacy policies.
