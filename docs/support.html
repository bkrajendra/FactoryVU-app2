Get help

FactoryVU is an industrial IoT companion app for monitoring and managing sensors and devices. For help with setup, connectivity, or troubleshooting, contact us with a short description of the issue.

  • Include device model + firmware version (if applicable)
  • Include your app version and iOS version
  • If local discovery fails, confirm the network (SSID) and whether multicast/Bonjour is allowed

Tip: The app uses an API key configured in Settings to load devices from the cloud.
