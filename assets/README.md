# App icon and splash source images

Place your source images here, then run:

```bash
npm run assets
```

(or `npx capacitor-assets generate`)

## Required files (minimal)

| File           | Size        | Format |
|----------------|-------------|--------|
| `icon-only.png` | **1024×1024** | PNG   |
| `splash.png`     | **2732×2732** | PNG   |

- **Icon:** Square app icon; keep it simple (looks good at 48px).
- **Splash:** Full-screen image; keep logo/content in the **center 1200×1200** (safe area).

See **docs/ICON_AND_SPLASH.md** for full steps and options.
